# discourse-onboarding-guide

Discourse plugin for onboarding newly registered users and users who join configured opt-in groups.

## Features

- Trigger onboarding for newly registered users and opt-in group members
- Versioned completion tracking with user custom fields
- Flagging tutorial with embedded post UI
- Pledge agreement step
- Notification preference selection (inline picker)
- Moderator PM demo step
- Tutorials step
- Persistent reopen bubble
